#### Description

Given a sorted linked list, delete all duplicates such that each element appear only _once_.

**Example 1:**

```
Input: 1->1->2
Output: 1->2
```

**Example 2:**

```
Input: 1->1->2->3->3
Output: 1->2->3
```

```python
def deleteDuplicates(self, head):
    p = head
    while p:
        while p.next and p.val == p.next.val:
            p.next = p.next.next
        p = p.next
    return head
```
