#### Description

Given an array of strings, group anagrams together.

**Example:**

```
Input: ["eat", "tea", "tan", "ate", "nat", "bat"],
Output:
[
  ["ate","eat","tea"],
  ["nat","tan"],
  ["bat"]
]
```

**Note:**
- All inputs will be in lowercase.
- The order of your output does notÂ matter.

```python
def groupAnagrams(self, strs):
    """
    :type strs: List[str]
    :rtype: List[List[str]]
    """
    groups = dict()
    for s in strs:
        k = ''.join(sorted(s))
        if k not in groups:
            groups[k] = [s]
        else:
            groups[k].append(s)

    return list(groups.values())
```

```python
def groupAnagrams(self, strs):
    groups = dict()
    for s in strs:
        count = [0] * 26
        for c in s:
            count[ord(c) - ord('a')] += 1
        k = '#'.join(map(str, count))
        if k not in groups:
            groups[k] = [s]
        else:
            groups[k].append(s)

    return list(groups.values())
```
