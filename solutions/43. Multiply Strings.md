#### Description

**Example 1:**

```
Input: num1 = "2", num2 = "3"

```

**Example 2:**

```
Input: num1 = "123", num2 = "456"
Output: "56088"
```

- Both `num1` and `num2`Â do not contain any leading zero, except the number 0 itself.

def multiply(self, num1, num2):
    if num1 == '0' or num2 == '0':
        return "0"

    ans = [0] * (len(num1) + len(num2))

    for j, b in enumerate(num1):
        b = ord(b) - 48
        for i, a in enumerate(num2):
            a = ord(a) - 48
            ans[i + j + 1] += a * b

    for i in range(len(ans) - 1, 0, -1):
        ans[i-1], ans[i] = ans[i-1] + ans[i] // 10, chr(ans[i] % 10 + 48)

    ans[0] = '' if ans[0] == 0 else chr(ans[0] + 48)
    return ''.join(ans)
